# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues by emailing **jhogendorn@gmail.com** with:

- A description of the vulnerability and its potential impact.
- Steps to reproduce or a minimal proof-of-concept.
- Any suggested mitigations, if you have them.

You will receive an acknowledgement within 48 hours. We aim to triage and issue a fix or
advisory within 14 days for confirmed vulnerabilities, depending on severity.

We will credit reporters in the release notes unless you request otherwise.
