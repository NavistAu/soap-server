Implement src/wsdl/resolver.rs:

Public types:
```rust
/// Output of WSDL resolution: the wired definition + all type information.
pub struct ResolvedWsdl {
    /// The original WsdlDefinition with QName strings (for WSDL serving)
    pub definition: WsdlDefinition,
    /// All types from inline and imported schemas, fully resolved
    pub type_registry: TypeRegistry,
    /// Original WSDL bytes (for serving on GET ?wsdl)
    pub raw_bytes: Vec<u8>,
}
```

The resolve algorithm:
```rust
pub fn resolve_wsdl(
    bytes: &[u8],
    loader: &dyn WsdlLoader,
    visited: &mut HashSet<String>,  // visited wsdl locations
) -> Result<ResolvedWsdl, WsdlError>
```

WsdlLoader trait (separate from SchemaLoader — loads WSDL files, not XSD):
```rust
pub trait WsdlLoader: Send + Sync {
    fn load(&self, location: &str) -> Result<Vec<u8>, WsdlError>;
}
```

Algorithm:
1. parse_wsdl(bytes) → WsdlDefinition (pass 1)
2. For each wsdl:import in WsdlDefinition.imports:
   - If location already in visited → skip (diamond import guard, per zeep pattern)
   - visited.insert(location)
   - loader.load(location) → bytes → recursively resolve_wsdl → merge operations/messages/bindings into root
3. Collect all inline xs:schema nodes (as roxmltree Document, re-parsed from raw bytes if needed)
4. For each schema: call xsd::parse_schema → RawSchema
5. Call xsd::resolve_schema on collected raw schemas → TypeRegistry
6. Wire cross-references in WsdlDefinition using TypeRegistry

Cross-reference wiring:
- For each binding: look up binding.port_type in definition.port_types map by QName
- For each operation in binding: look up input/output message in definition.messages map
- Unresolvable QName → WsdlError::UnresolvedRef(qname_str)

rewrite_wsdl_address(bytes: &[u8], new_url: &str) → Vec&lt;u8&gt;:
- Use quick-xml to stream through the WSDL, find soap:address or soap12:address elements
- Replace their location= attribute value with new_url
- All other content passes through unchanged
- Return the modified XML bytes

```rust
pub use resolver::{resolve_wsdl, ResolvedWsdl, WsdlLoader, rewrite_wsdl_address};
```
