Default mount path: "/soap" — configurable via `.path("/soap")`.

Request pipeline (axum handler function — extract State<Arc<SoapService>>, HeaderMap, Bytes):
```
1. detect_soap_version(content_type_header)
   → Err: return 500 + fault XML
2. parse_envelope(body_bytes)
   → Err: return 500 + fault XML
3. Extract body_first_child QName using quick-xml (small helper: just extract the QName without full parse)
   → Err: return 500 + fault XML
4. dispatch::route(&table, &qname, soap_action_header)
   → Err: return 500 + fault XML
5. If entry.auth_required:
   a. Find wsse:Security header in envelope.header_children
      → None: return 500 + SEC-07 fault
   b. wssec::validate_username_token(security_bytes, auth_fn, nonce_cache, tolerance, now)
      → Err: return 500 + fault XML
6. dispatch::validate_request(body_bytes, type_registry, entry.input_type)
   → Err: return 500 + fault XML
7. entry.handler.handle(body_bytes).await
   → Err(fault): return 500 + fault XML
   → Ok(response_bytes): continue
8. serialize_envelope(response_bytes, soap_version)
9. return 200 with correct Content-Type
```

WSDL handler (axum handler for GET):
```
1. Check Query params for "wsdl" key → if absent: return 404
2. Build server URL from Host header (or X-Forwarded-Host): "http://{host}{mount_path}"
3. rewrite_wsdl_address(&wsdl_raw, &server_url) → modified bytes
4. return 200 with Content-Type: text/xml
```

Helper to extract body first-child QName from ParsedEnvelope.body_element bytes (reuse quick-xml NsReader, just read the first start element and return its resolved QName).
